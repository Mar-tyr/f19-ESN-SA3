Please read through the following information before you proceed.

You will able to share your status in the ESN. The categories of status are the following:

Status Explanation Color Code Possible Icon
OK I am OK: I do not need help. Green
Help I need help, but this is not a life threatening emergency. Yellow
Emergency I need help now: this is a life threatening emergency! Red
Undefined The user has not proved any status info. N/A

Please acknowledge if you have read the above information and understood the corresponding explanations.
